Home Screen

The home screen is displayed when no videos are loaded. Tap ADD VIDEO or the + icon in the toolbar to open the file picker.

Toolbar

  • Grid SelectorTap the grid label (e.g. "1×1 - Single") in the top-left to choose a layout. Up to 6 videos can be played simultaneously.
  • FD StatusDisplays the current app health status: Healthy / Caution / Warning. This reflects the number of open file descriptors. If it shows Warning, consider closing some videos.
  • +Opens the file picker screen.
  • RotateToggles the screen orientation between portrait and landscape.
  • ☰Opens the menu (Settings / Manual / About).

File Picker

Browse and select videos from your device storage or connected USB drive. Internal and USB videos are displayed together in a single list.

Toolbar

  • ✕Close the file picker and return to the home screen.
  • RELOADRe-scan and refresh the video file list.
  • SortSort the list by Name, Type, Size, or Date Modified. Tap the sort direction to toggle ASC / DESC.

Video List

  • Each item displays a thumbnail, file name, and a badge showing its source: Internal or USB.
  • Tap a video to select it. Multiple selections are supported.

Bottom Bar

  • Internal / USB / TotalShows the number of videos found in each source.
  • CANCELClose the file picker without adding videos.
  • CLEARDeselect all selected videos.
  • ADD FILESAdd the selected videos to the current grid and start playback.

Playback Controls

Tap on a playing video to show the overlay controls. The overlay automatically hides after 2 seconds.

Top Bar

  • File NameDisplays the currently playing file name.
  • ✕Close this video. If all videos are closed, the app returns to the home screen.

Playback Buttons

  • |◀Play the previous video in the list.
  • ▶ / ❚❚Play or pause the current video.
  • ▶|Play the next video in the list.

Bottom Controls

  • ×1Playback speed selector (×0.25 / ×0.5 / ×1 / ×1.5 / ×2).
  • 🔊Toggle mute on/off for this video.
  • ♥Add or remove the current video from your favorites.
  • Fav ShuffleReplace this slot with a random video from your favorites.
  • ShuffleReplace this slot with a random video from all available videos (internal + USB).
  • RotateRotate the video 90° clockwise.
  • ResizeCycle through display modes: Fit / Fill / Zoom.

Multi-Video Playback

When multiple videos are playing, swipe up or down on the video area to show the global controls.

Toolbar (shown on swipe)

  • Grid SelectorChange the grid layout while videos are playing.
  • FD StatusDisplays current app health. If it shows Warning, consider reducing the number of active videos.
  • +Add more videos to available slots.
  • RotateToggle screen orientation.
  • ☰Open the menu.

Bottom Controls (shown on swipe)

  • ❚❚ PAUSE ALLPause all currently playing videos.
  • ▶ PLAY ALLResume playback of all videos.
  • 🗑 CLEAR ALLStop and remove all videos. A confirmation dialog will appear.

Settings

Open Settings from the ☰ menu in the top-right corner.

Search Folders

  • DownloadInclude the Download folder when scanning for videos.
  • MoviesInclude the Movies folder when scanning for videos.

Search Files

  • Show ThumbnailsEnable or disable thumbnail generation in the file picker. Turn this OFF on low-end devices if loading is slow.
  • Exclude Hidden FilesHide files whose names start with a dot (.) from the file picker.
